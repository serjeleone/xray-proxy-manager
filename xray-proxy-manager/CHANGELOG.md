# Журнал изменений

## 0.5.0

- Добавлены два независимых Xray-слота `xray-a` и `xray-b`.
- Параметры SOCKS5 разделены по слотам: `socks_tcp_a`, `socks_tcp_b`, `socks_udp_a` и `socks_udp_b`.
- Добавлен резервный запуск и предварительная проверка нового outbound.
- Добавлено переключение совместимого внешнего selector через HTTP API.
- Добавлен контроль соединений и безопасное завершение прежнего слота.
- Добавлен автоматический откат после неуспешного переключения.
- Добавлено восстановление состояния после временной недоступности selector API.
- Добавлены автоматическая проверка, выбор лучшего outbound и обновление подписки.
- Добавлены CI-сборка и публикация multi-arch образа для архитектур `amd64` и `aarch64`.
- Исправлена компоновка заголовка активного outbound после удаления устаревшего элемента управления.
- Добавлен просмотр изменений текущего релиза по нажатию на номер версии.
- Исправлены открытие окна логов и вертикальное выравнивание кнопки «Логи».
- Уточнён подсчёт активных соединений дренируемого слота без двойного учёта TCP и состояний `TIME_WAIT`.
